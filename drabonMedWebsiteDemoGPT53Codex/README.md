# Drabon Med redesign demos

This workspace contains multiple vanilla HTML/CSS/JS redesign concepts for the Drabon Med website.

## Included

- `index.html` – gallery page to access each demo concept
- `shared/content.js` – normalized content model sourced from the provided Hungarian text files
- `demos/demo-1/` – Concept 1: single-page modern healthcare landing page
- `demos/demo-2/` – Concept 2: multi-page corporate healthcare website
- `demos/demo-3/` – Concept 3: editorial premium single-page website
- `demos/demo-4/` – Concept 4: senior-friendly ultra-clear single-page website
- `demos/demo-5/` – Concept 5: high-trust healthcare brochure single-page website
- `demos/demo-6-red-multipage/` – Bonus concept: red-accented medical multi-page website

## Notes

- Copy is kept in Hungarian for stakeholder review.
- Code, structure, and component naming are in English.
- The distance price calculator is mocked for now and ready to be replaced later with a map API.

## Current concepts

### Demo 1

Single-page direction with softer visuals, more motion, and landing-page style storytelling.

### Demo 2

Multi-page direction with separate `services`, `pricing`, and `contact` pages for a more traditional company website structure.

### Demo 3

Premium editorial single-page direction with a warmer palette, slower storytelling rhythm, and more trust-building presentation blocks.

### Demo 4

Senior-friendly single-page direction with larger typography, stronger contrast, bigger buttons, and the shortest path to calling, reading key services, and opening the calculator.

### Demo 5

Formal brochure-style healthcare direction with stronger trust framing, clearer professional presentation, and visible legal/context blocks for a more established company feel.

### Bonus red multi-page concept

Additional multi-page direction with a more clinical red accent palette for a stronger classic medical visual signal.

## Open locally

Open `index.html` directly in a browser, or serve the folder with any static server for best results.
