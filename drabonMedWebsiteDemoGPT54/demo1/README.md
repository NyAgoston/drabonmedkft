# Drabon Med redesign demo

This is a vanilla HTML, CSS, and JavaScript redesign demo for the Drabon Med website.

## Included

- modern one-page responsive layout
- Hungarian content adapted from the `text/` folder
- visible mocked distance price calculator in HUF
- mobile navigation and scroll reveal animations
- provided image assets (`logo.png`, `car.jpg`, `badge.jpg`)

## Open the demo

Open `index.html` directly in a browser, or run it from any simple static file server.

## Notes

- The price calculator is intentionally mocked for demo purposes.
- A future map API integration can replace the local distance matrix in `script.js`.
- Code is written in English-friendly file and variable names while the UI remains Hungarian.
