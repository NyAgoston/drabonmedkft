# Drabon Med redesign demo 3

This is the third redesign concept for the Drabon Med website.

## Direction

This option is the most image-led and presentation-focused concept so far.

## Included

- responsive one-page layout
- bold modern hero section
- image-driven cards and floating content blocks
- Hungarian UI copy based on the provided text content
- visible mocked HUF price calculator
- shared image assets from the project root

## Notes

- The calculator is still a demo and uses mocked route distances.
- This concept is meant to feel more dynamic and visually memorable than `demo1` and `demo2`.
