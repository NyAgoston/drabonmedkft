# Drabon Med redesign demo 2

This is the second redesign concept for the Drabon Med website.

## Direction

This option uses a lighter, more clinical and trust-focused visual style than `demo1`.

## Included

- responsive one-page layout
- Hungarian UI copy based on the provided content files
- visible mocked HUF price calculator
- light medical-style design system
- shared image assets referenced from the project root

## Notes

- The calculator is mocked for demo use.
- Real route pricing can later replace the local distance object in `script.js`.
- This concept is meant to feel clearer, brighter, and more institutional than `demo1`.
