# Drabon Med redesign demo 4

This is the fourth redesign concept for the Drabon Med website.

## Direction

This concept is the most accessibility-first and easy-to-read light mode option.

## Included

- responsive one-page layout
- larger type and calmer spacing
- simplified, low-clutter content structure
- visible mocked HUF calculator
- shared image assets from the project root

## Notes

- The goal of this concept is clarity and comfort, especially for older visitors.
- The calculator remains mocked and can later be connected to a real route API.
